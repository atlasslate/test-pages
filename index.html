Sol Slate v0.2.0

Official firmware flasher for the Heltec Wireless Paper. Use Chrome or Edge via USB connection.

Pre-flight Check:
1. Connect Sol Slate via USB-C.
2. If it isn't recognized, hold the BOOT button while plugging it in.

Troubleshooting

Device not found?

Ensure you are using a data-capable USB cable. Some "charging only" cables won't work.

Stuck in a boot loop?

This version uses the merged.bin which includes the bootloader. If it still loops, try clicking "Erase Device" inside the install menu before flashing.
